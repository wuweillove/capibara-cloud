# Capibara Cloud

Interfaz web para OpenClaw.

## Requisitos

- Docker y Docker Compose

## Instalación

1. Clona este repositorio
2. Ejecuta `docker-compose up -d`
3. Abre http://localhost:3000 en tu navegador

## Uso

1. Ingresa tu API key de Google o OpenAI
2. Selecciona el modelo a usar
3. Haz clic en "Iniciar"
4. Interactúa con el chat

## Notas

Esta aplicación requiere al menos 2GB de RAM para funcionar correctamente.
